Mein Webentwicklungsprojekt
Von der Idee zur Website

Dieses Projekt basiert auf den drei Grundpfeilern der modernen Webentwicklung: HTML, CSS und JavaScript. Ich habe mit dem HTML Grundgerüst begonnen, um die Struktur und den Inhalt der Seite zu erstellen.

Anschließend wurde mit CSS das Layout gestaltet, inklusive responsivem Design und ansprechender Typografie.

Ziel des Projekts

Dieses Projekt diente dazu, die Grundlagen der Webentwicklung mit HTML und CSS zu erlernen. Ziel war es, eine moderne, responsive Website ohne Baukastensysteme wie WordPress zu erstellen.

  1. Grundlagen in HTML und CSS praktisch anwenden
  2. Eine eigene, strukturierte Webseite vollständig selbst erstellen
  3. Vorbereitung auf eine Weiterbildung im Bereich Webentwicklung

Was ist Omnifood?

Omnifood ist ein fiktives Startup, das sich auf KI-gestützte, gesunde Essenslieferung spezialisiert hat.
Dieses Projekt wurde als realitätsnahes Webdesign-Training entwickelt und diente dazu, eine professionelle, responsive Landingpage zu erstellen.
Ziel war es, moderne HTML- und CSS-Techniken wie Flexbox, Grid und Media Queries praxisnah anzuwenden.
Das Projekt simuliert eine echte Unternehmensseite, inklusive Navigationsstruktur, responsivem Design und einem klaren Fokus auf Benutzerfreundlichkeit.

  • Verwendung von Flexbox und Grid für Layouts
  • Gestaltung ansprechender und lesbarer Inhalte
  • Saubere Strukturierung mit wiederverwendbarem CSS
